# CacheMemories

CacheMemories is a program that implements a cache memory simulation.

This version supports the LRU, MRU and LFU cache types and the Hit/Miss function.

## Usage

Compile using

```
mvn compile
```

Create a jar using

```
mvn package
```

Run main using

```
java -jar target/cache.jar
```

Run unit tests using

```
mvn test
```
